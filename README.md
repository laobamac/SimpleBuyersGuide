# 简介

什么是「黑苹果硬件避坑指南」？为什么你需要关注这篇文章？  
这是一份关于组装 Hackintosh 时**不应购买**的硬件指南，并将随着新硬件的发布持续更新。虽然本指南不会给出具体的产品推荐，但会帮助你避开雷区，指明正确方向。
